About This Website

This Task Manager website helps users organize and manage their daily tasks.
Users can view tasks and add new ones using a simple and accessible interface.

README - Accessibility and Semantic Decisions

1. Use of Semantic Elements

Semantic elements like header, nav, main, section and footer were used to clearly define the structure of the page.
This helps screen readers understand the layout and improves accessibility.

2. Skip to Content Link

The skip to content link allows screen reader users to skip the navigation menu and go directly to the main content,
making navigation faster and more accessible.

3. Proper Form Labels

Each form input uses a label with a "for" attribute connected to the inpu.
This improves usability and accessibility, especially for screen reader users.
